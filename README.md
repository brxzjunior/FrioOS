# FrioOS
❄️ FrioOS – Sistema de Ordens de Serviço para Técnicos de Refrigeração  FrioOS é um sistema web desenvolvido para ajudar técnicos de refrigeração e microempreendedores (MEI) a organizar seus atendimentos, clientes e serviços de forma digital, substituindo anotações em papel e mensagens desorganizadas.
